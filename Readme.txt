1. Download ODM from http://sourceforge.net/projects/opendecisionmak/
2. Install 32 bit JDK
3. Download 32 bit eclipse.
4. The project requires few jar files to compile. All the Jar files with which the project was compiled are in the libs folder. the jar files needed for compiling the project are (there are many in the folder but don't really need to add all of them unless you are doing something fancy like PDF printing)
	org.eclipse.swt.win32.x86_3.6.1.v3655c.jar
	org.eclipse.jface_3.6.1.M20100825-0800.jar
	jdom.jar
	jasterreports-3.7.5.jar
